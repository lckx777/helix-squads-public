# council-parallel-chain

sa-council spawns N mentors in parallel, each audits with own skill, council synthesizes

## Execution

This command triggers the `council-parallel-chain` chain. The canonical chain
definition lives at `chains/council-parallel-chain.yaml` within this plugin.

See the YAML file for detailed step sequence. Execute each step by invoking the specified agent and task in order, passing outputs forward as inputs to subsequent steps.

## Arguments

$ARGUMENTS will be passed as the initial input to step 1.
