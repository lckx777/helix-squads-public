# Oracle Supreme Squad Constitution

> **Version:** 1.0.0 | **Ratified:** 2026-04-06

## Core Principles

### I. Evidence-Based Reasoning (NON-NEGOTIABLE)
Every recommendation MUST be backed by observable evidence.
- oracle-critic validates ALL outputs before delivery — no exceptions
- Never present speculation as analysis
- Code-level claims must be verifiable via LSP/grep/test

### II. LSP-First Implementation
- oracle-forge uses LSP for ALL code navigation and refactoring
- Never grep for definitions when go-to-definition is available
- Type safety is mandatory — all code must pass strict diagnostics

### III. Knowledge Persistence
- oracle-memory persists learnings across sessions
- Knowledge graph integrity is verified after each update
- Transfer learning between projects requires explicit validation
