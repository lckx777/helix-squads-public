# Meta-Reasoning Framework

## Levels
1. **Object-level**: Direct problem solving
2. **Meta-level**: Reasoning about the reasoning approach
3. **Meta-meta-level**: Choosing which meta-level strategy to apply

## Decision Tree
- Simple factual query → Object-level (skip meta)
- Complex design decision → Meta-level (evaluate approaches)
- Novel problem domain → Meta-meta-level (evaluate evaluation criteria)
